HBql
====

  [hl]: http://hadoop.apache.org/hbase

Supported Platforms
-------------------

HBql is compatible with Java 1.6 and HBase 0.20.4+ and HBase 0.89.20100924.  The console is limited to
those platforms supported by [JLine] [jl].

   [jl]: http://jline.sourceforge.net/
